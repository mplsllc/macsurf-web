The modern web, on a 25-year-old Mac.

MacSurf is a real web browser for Classic Mac OS 9. Real CSS3. Real ES5 JavaScript. Real PNGs with alpha. Rendering live inside the 64 MB memory floor of a beige G3.

You are viewing the plain HTTP edition, served without modern CSS or scripting so that Netscape 3, Internet Explorer 5, MacWeb, iCab 2, and other vintage browsers can read it. The full site is at https://macsurf.org/ for modern browsers.

The web outgrew Classic Mac OS twenty years ago.

Every browser project for OS 9 quietly stopped: Internet Explorer in 2003, iCab not long after, Classilla in 2014. Modern sites became unreachable from the machine they were designed on. MacSurf is the answer -- a NetSurf port built from scratch in CodeWarrior against the Carbon API, with a real rendering pipeline and a tiny Go proxy for TLS.

Three pieces. One working browser.

1. MacSurf (the browser). A native Carbon application written in C with CodeWarrior 8 Pro. Fork of NetSurf, with QuickDraw plotters, GWorld back-buffer, libcss, Duktape ES5 JavaScript, and lodepng for real alpha-blended PNG.

2. macSSL (the proxy). A single self-contained Go binary that fronts HTTPS for the browser. No config files, no dependencies. Point the browser at the host and you are on the modern web.

3. The cross-dev workflow. Source stays cross-compile-clean against Retro68 PowerPC GCC, so syntax checks and clean builds happen on Linux. CodeWarrior on the Mac produces the shipping PEF binary.

A full rendering engine, not a demo.

Rendering pipeline

  • Full NetSurf fetch, parse, cascade, layout, plot
  • Native libcss with var() resolution
  • QuickDraw plotters with an offscreen GWorld back-buffer
  • CSS 2.1 painting order and z-index stacking contexts

CSS -- 150+ properties

  • Custom properties and var()
  • Flexbox -- justify-content, order, align
  • Grid V1 -- grid-template, gap
  • Gradients, border-radius, box-shadow, opacity
  • transform, text-shadow, counters, aspect-ratio

JavaScript -- Duktape ES5

  • Closures, prototypes, regex, JSON
  • Promises via polyfill, deep recursion
  • Live Mandelbrot computed in-page
  • About 6 seconds for ackermann(3,7) on a 233 MHz G3

Images -- all five formats

  • PNG with real per-pixel alpha (lodepng + CopyMask)
  • GIF with palette transparency
  • JPEG, BMP and TIFF decoding
  • QuickTime Graphics Importers as the content handler

Networking

  • Open Transport TCP, plain non-InContext calls
  • HTTP/1.1 with chunked, keep-alive, and 3xx follow
  • Connection pooling, 15 second no-progress timeout
  • HTTPS via the Go proxy or native macSSL

Browser chrome

  • Address bar, back, forward, reload, home
  • Status bar, page-info, multi-window
  • Smooth scroll bar and keyboard scrolling
  • Platinum-themed native Carbon UI

Two ways in.

Build the browser on the Mac with CodeWarrior:

  open MacSurf.mcp
  make --target PowerPC

Run the proxy with one Go binary:

  cd proxy
  go build -o macsurf-proxy
  ./macsurf-proxy
  -> listening on :8080

Download

Two StuffIt archives. Right-click (or hold the mouse down) and choose Save Target As… / Save Link As… to download. Decompress with StuffIt Expander on the Mac.

  • MacSurf.sit -- ready-to-run binary, 567 KB. Expand on Mac OS 9.1 or newer with CarbonLib 1.5+ and double-click to launch.
  • MacSurf-BuildPack.sit -- full source + CodeWarrior 8 project file, 4.0 MB. Expand, open MacSurf.mcp in CodeWarrior, choose Build.

If your browser opens the file as text instead of downloading, hold the Option key while clicking the link (Classic Mac), or use the right-click save option in modern browsers.
